Images

General Guidelines

  • Use SVG whereever possible, this ensures a crisp image on every device and a really small file size.
  • Use sprites to reduce network requests.
  • Use inline SVG when you need to dynamically change colors.
  • Use inline SVG sprites for themed icon systems.
  • Keep image names lowercase and hyphenated and ensure they make sense.
  • Use picturefill to serve appropriatley sized images to difference screen sizes.
  • For raster images, always save retina versions saved at a lower quality, and tell picturefill that they are 2x versions.
  • Always use JPG, unless trancparency is required, then use PNG.

Folder Structure

Here is the folder structure we follow for our project:

  • app
    • data
    • handlebars
    • images
      • content
      • interface
    • javascript
      • modules
        • exampleModule.js
      • application.js
      • config.js
    • stylesheets
      • blocks
        • block-name.sass
      • generic
        • buttons.sass
        • typography.sass
      • layouts
        • homepage.sass
      • modules
        • header.sass
      • settings
        • colors.sass
        • fonts.sass
      • tools
        • bourbon
        • functions
        • mixins
    • vendor

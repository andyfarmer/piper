Introduction

Welcome to Primer, the Rawnet UI Styleguide.

The aim of this styleguide is to ensure consistent coding practices across all developers in the team, which in turn will enable all developers to easily pick up and maintain any project. Every line of code should appear to be written by a single person, no matter the number of contributors.

You should do your best to make sure any code you contribute is in a fit state for the next developer. We encourage readabilty and clarity above abstraction and cleverness.
